# Global Sales Data Analytics Project

## Project Overview
This project is an end-to-end data analytics workflow focused on analyzing global sales data. By transforming raw sales data into meaningful business insights, the project helps organizations identify key trends, customer behaviors, product performance, and geographical patterns to make data-driven decisions.

The workflow spans across data cleaning, exploratory analysis, SQL querying, and interactive dashboard development.

## Business Objectives
The key objectives of this project are:
- Analyze overall sales performance and revenue trends
- Identify top-performing products and categories
- Understand customer purchasing behavior
- Analyze regional and geographic sales performance
- Identify monthly and seasonal sales patterns
- Build an interactive dashboard for business decision-making

## Tools and Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI (Power Query, DAX)
- Jupyter Notebook

## Project Pipeline

### 1. Data Cleaning and Preparation
The raw dataset was cleaned and prepared for analysis using Python. This involved data quality checks, handling missing values, removing duplicate records, correcting data types, and formatting dates.

### 2. Exploratory Data Analysis
Exploratory data analysis was performed using Python to understand patterns, trends, and relationships within the sales data. The analysis covered sales distribution, category performance, regional contributions, and customer segments.

### 3. Business Analysis Using SQL
SQL was used to query the cleaned dataset and answer critical business questions. Key focus areas included calculating overall revenue, finding top-performing products, comparing regional sales, and evaluating seasonal trends.

### 4. Power BI Dashboard Development
An interactive dashboard was developed using Power BI to visualize key insights and provide business-friendly reporting. The report is divided into four main pages:

- **Executive Dashboard:** Provides a high-level overview of total revenue, orders, quantity sold, and average order value.
- **Sales Trend Analysis:** Tracks monthly and quarterly revenue streams to identify growth patterns over time.
- **Product Performance:** Focuses on catalog optimization by analyzing category contributions and product-wise revenue.
- **Customer and Geographic Analysis:** Maps consumer behavior matrices against regional operational performance.

## Key Insights Generated
The analysis successfully isolated the highest revenue-generating products and identified the top geographic sectors driving sales. It also revealed critical monthly and seasonal purchasing patterns to help improve future business decisions and inventory planning.
